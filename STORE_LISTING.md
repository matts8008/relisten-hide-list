Copy/paste these into the Chrome Web Store developer dashboard when you submit.

## Name
Relisten Hide List

## Summary (max 132 chars)
Auto-skip Relisten tracks by keyword. Toggle hide words on/off and hide or unhide songs straight from the track list.

## Category
Suggested: **Entertainment** (or **Productivity**)

## Description
Relisten Hide List makes long Relisten shows play the way you want.

Tell it which songs you don't want to hear, and it quietly skips them for you — no more reaching for the Next button.

FEATURES
• Hide list — add any word or phrase. Any track whose title matches gets skipped automatically.
• Reversible toggles — uncheck a word to turn it off without deleting it. Turn it back on anytime.
• Hover to Hide — hover a track in the show's track list and click "Hide" to add that exact song.
• One-click Unhide — hidden tracks are struck through with an "Unhide" button. Click it and the extension turns off the exact word that was hiding the song.
• Wildcards — use * for any text and ? for a single character.
• Visual marks — skipped songs are dimmed and struck through so you always know what's on the chopping block.
• Works from the page or the toolbar popup — your list is the same in both.

PRIVACY
No accounts, no tracking, no ads, no data leaves your browser. The extension only runs on relisten.net and only stores your hide list via Chrome sync. See the privacy policy for details.

## Privacy practices (dashboard answers)
- Does your extension collect user data? **No.**
- Permission justification:
  - `storage`: Save the user's hide list and enabled/disabled settings.
  - Host permission `relisten.net`: Read the currently playing track and track list to skip and mark tracks on that site.
- Single purpose: Automatically skip and manage Relisten tracks based on the user's keyword hide list.
- Remote code: **No remote code used.**
