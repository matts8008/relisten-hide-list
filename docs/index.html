Relisten Hide List

A Chrome extension that lets you skip songs on Relisten. Add words to your hide list — like tuning, drums, or space — and matching songs are skipped automatically. Your list saves across every Grateful Dead show.

Download View on GitHub Chrome Web Store — coming soon

How to use

  • Click the Hide pill by the player and add a word — try tuning, drums, or space.
  • Any song whose title matches is skipped and struck through.
  • Uncheck a word to turn it off without deleting it.
  • Hover any track to Hide or Unhide it on the spot.

Install

  1. Download the ZIP and unzip it.
  2. Open chrome://extensions and turn on Developer mode.
  3. Click Load unpacked and select the unzipped folder.
  4. Open a Relisten show and start playing.
